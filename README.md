# RsFtch

##### _Lightning fast hardware fetch written in rust._

### Screenshots

### Supported distros
- Arch Linux
- Debian
- Fedora
- Endeavour OS

### Installation
- `git clone https://github.com/charklie/rsftch.git ~/rsftch/`
- `cd rsftch`
- `cargo install --release`
- `export PATH=$PATH:~/.cargo/bin/`

### Usage
- `rsftch`
